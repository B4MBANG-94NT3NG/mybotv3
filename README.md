## Penginstalan

Jika pertama kali membuka termux lakukan perintah berikut :
```bash
> pkg install git
> git clone https://github.com/miqbal-22/mybotv3
> ls
> cd mybotv3
> bash install.sh
```
Jika mengalami error, lakukan perintah berikut :
```bash
> npm audit fix
```
Jalankan bot
```bash
> node index.js
```
Terakhir scan kode QR
